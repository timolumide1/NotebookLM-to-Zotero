# Privacy Policy - NotebookLM to Zotero Extension

**Last Updated:** February 16, 2026

## Data Collection
This extension does NOT collect any user data.

## Data Storage
- Your Zotero API key is stored locally in your browser only
- Extension settings are stored locally using Chrome's storage API
- No data is sent to any servers except Zotero's official API

## Data Transmission
Extension communicates ONLY with:
1. NotebookLM (to read source data)
2. Zotero API (to export citations)
3. CrossRef/arXiv/PubMed APIs (for metadata enrichment)

No analytics, no tracking, no third-party services.

## Your Rights
- All data is stored locally - you control it
- Delete extension = delete all stored data
- No accounts, no servers, no tracking

## Contact
For questions or concerns, please [open an issue](https://github.com/timolumide1/NotebookLM-to-Zotero/issues).
